1 — 概要
位置づけ

Hermes Agent は Nous Research が公開する Python 製の AI エージェント。単発のチャットではなく、スキルの自動生成・改善、永続メモリと会話検索、マルチチャネル Gateway を一体で持つ。モデルプロバイダは固定されず、hermes model で切り替え可能。

2 — 学習とメモリ
閉ループ

README が強調するのは「経験から学ぶ」一連の流れだ。複雑タスクのあとにスキルを自動生成し、利用中に内容を改善する。メモリはエージェント側でキュレーションされ、必要に応じてナッジされる。

  • セッション検索: FTS5 ベースの検索に LLM 要約を組み合わせ、過去会話を引き戻す。
  • ユーザモデル: Honcho による弁証法的ユーザモデリング（README 記載）。
  • 標準: agentskills.io オープン標準との互換。
タスク実行 → スキル生成 / 更新 → メモリ書き込み → 次セッションで検索・再利用
研究向け

バッチ軌道生成、Atropos RL 連携、軌道圧縮など、ツール呼び出しモデル向けの研究機能も README に列挙されている。

3 — CLI とメッセージング Gateway
二つの入口

対話の起点は ターミナル UI（hermes）か、Gateway（hermes gateway）経由のメッセージングのどちらか。起動後は多くのスラッシュコマンドが共通。

操作CLIメッセージング
会話開始hermesgateway setup → start → ボットへ送信
新規スレッド/new / /reset同左
モデル変更/model [provider:model]同左
スキル/skills / /<skill>同左
中断Ctrl+C または新規メッセージ/stop または新規メッセージ
プラットフォーム: Telegram、Discord、Slack、WhatsApp、Signal、メールなど。音声メモの文字起こしや、チャネル間の会話連続性も README の機能表に含まれる。

5 — クイックスタート
インストール

公式インストーラは Linux / macOS / WSL2 向け。ネイティブ Windows は非対応（WSL2 推奨）。

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
source ~/.bashrc   # または source ~/.zshrc
hermes

よく使うコマンド

hermes              # 対話 UI
hermes model        # プロバイダ・モデル選択
hermes tools        # ツール有効化
hermes gateway      # メッセージング Gateway
hermes setup        # セットアップウィザード
hermes doctor       # 診断
OpenClaw からの移行

hermes claw migrate で設定・メモリ・スキル・API キー等をインポート可能。README にプリセットと --dry-run が記載されている。
