Modern Web Guidance

GoogleChrome/modern-web-guidance — コーディングエージェントに モダン Web プラットフォームの専門知識を注入する SKILL.md + CLI。Google Chrome / Microsoft Edge チームとコミュニティが支援。README と SKILL.md から要約。

解決する問題

LLM の学習データにはレガシー実装が多く、エージェントは 肥大化した JavaScript で、すでにネイティブ API がある課題を実装しがちです。Modern Web Guidance は、トークン効率を意識した 実運用向けガイドをコンテキストへ返し、パフォーマンス・アクセシビリティ・適切なフォールバックを揃えます。

HTML/CSS とクライアント JS のタスクでは、SKILL.md が 最初に実行することを要求します（View Transitions、Popover、INP、Passkeys、Built-in AI など）。

2 — CLI の流れ（search → retrieve）
SKILL.md の手順

エントリは npm パッケージ modern-web-guidance（bin → modern-web.mjs）。実装の詳細は modern-web-guidance-src、公開用リポジトリはスキル配布専用です。

Step 1 — search

ユースケース検索
npx -y modern-web-guidance@latest search "animate a dialog modal backdrop"

返却は JSON 配列（id、description、category、featuresUsed、tokenCount、similarity）。曖昧なときは list で全ガイドを一覧できます。

Step 2 — retrieve

ガイド本文の取得
npx -y modern-web-guidance@latest retrieve "animate-to-from-top-layer"

複数 ID はカンマ区切り。出力は Markdown の実装手順。フレームワーク非依存なので、React / Vue / Angular へはプロジェクトに合わせて適用します。

ブラウザサポート: Baseline Widely available はフォールバック不要が既定。それ以外はガイドの推奨に従い、ユーザーがブラウザ方針（例: Safari 17.4+、ポリフィル禁止）を定義していればそれを優先します。

3 — カバーする分野
User Experience

View Transitions、スクロール駆動アニメ、:has()、ダークモード、パララックス、3D シーン内 UI など。

CSS Layout

コンテナクエリ、subgrid、oklch、text-wrap、行高トリミング。

Performance

LCP / INP、content-visibility、Fetch Priority、scheduler.yield、長タスク分割。

Forms & UI

Anchor Positioning、Popover API、:user-invalid、オートフィル、カスタム <select>。

Accessibility

エラー告知と aria-invalid の同期、キーボードフォーカス管理。

Built-in AI / WebMCP

オンデバイス翻訳・要約・言語検出、宣言的 WebMCP（エージェント向けフォーム）。

ガイド例: navigation-drawer（README 参照）。

5 — インストール
代表的な方法
対話式ウィザード（推奨）
npx modern-web-guidance@latest install

インストールせず試す場合:

検索・取得のみ
npx modern-web-guidance@latest search "…"
npx modern-web-guidance@latest retrieve "guide-id"

その他

  • 更新: npx modern-web-guidance@latest update（install 経由の場合）
  • Claude Code: /plugin marketplace add GoogleChrome/modern-web-guidance など（README 参照）
  • テレメトリ無効: シェルに DISABLE_TELEMETRY=1（収集内容は serving 実装を参照）
Windows: npx が失敗する場合は npx.cmd。オフライン時は npx --offline を試します。
