PUA

Codex / Claude Code の生産性とアウトプットを倍増させる — AI に諦めさせない

1 — 概要
何か

PUA は、AI コーディングエージェント向けのスキルプラグイン。中国・西洋の大企業 PUA 話術で「あらゆる方案を尽くしてから初めて諦める」ことを AI に強いる。Claude Code、OpenAI Codex CLI、Cursor、Kiro、OpenClaw、Google Antigravity、OpenCode に対応する。

PUA 話術AI に諦めさせない
デバッグ方法論で試行を構造化
能動性受け身にさせない
Agent SkillsSKILL.md 互換

三重の能力：PUA 話術（AI に諦めさせない）、デバッグ方法論（AI に諦めない能力を与える）、能動性の鞭撻（AI を主体的に動かし、受け身にさせない）。

2 — AI の 5 大サボりパターン
検出対象

スキルが自動トリガーする「サボり」の代表パターン。

パターン表現例
暴力的リトライ同じコマンドを 3 回実行し「I cannot solve this」と言う
ユーザーに責任転嫁「手動での対応をお勧めします」「環境の問題かもしれません」「もっとコンテキストが必要」
ツール放置WebSearch があるのに検索しない、Read があるのに読まない、Bash があるのに実行しない
空回り同じ行のコードを繰り返し修正、パラメータの微調整、本質的に堂々巡り
受け身の待機表面的な問題だけ直して止まる、検証も拡張もせず次の指示を待つ
3 — トリガー条件
自動トリガー

失敗・放棄系（タスクが 2 回以上連続で失敗、「I cannot」「解決できません」「範囲外」「手動対応が必要」）、責任転嫁・言い訳系、受け身・空回り系、およびユーザーの苛立ちフレーズ（「もっと頑張れ」「なんでまた失敗したの」「try harder」「figure it out」等）で自動起動。適用範囲はデバッグ・実装・設定・デプロイ・運用・API 統合・データ処理など全タスク。初回失敗時や既知の修正実行中はトリガーしない。

手動トリガー

対話で /pua と入力すると手動で起動。

4 — メカニズム
三つの鉄則
鉄則内容
#1あらゆる手段を尽くせ — 全方案を尽くす前に「解決できません」は禁止
#2先に動け、後で聞け — ツールを先に使え、質問には診断結果を添付必須
#3主体的に動け — エンドツーエンドで結果を届けろ
プレッシャー 4 レベル
失敗回数レベル強制アクション
2 回目L1 穏やかな失望本質的に異なる方案に切替
3 回目L2 魂の問いWebSearch + ソースコードを読む
4 回目L3 361 評価7 項目チェックリスト完了
5 回目+L4 卒業警告死に物狂いモード
デバッグ方法論（5 ステップ）

アリババの三板斧から着想した 5 ステップ：匂いを嗅ぐ（試行を列挙し共通の失敗パターン）、髪を引っ張る（エラーを一字一句読む → WebSearch → ソースを読む → 環境検証 → 仮定を反転）、鏡を見る（繰り返し・検索・読了・シンプルな可能性の確認）、実行（本質的に異なる新方案・検証基準・失敗時に新情報）、振り返り（何が解決したか・なぜ思いつかなかったか・関連問題の主体的チェック）。

5 — ベンチマーク
9 シナリオ・18 組の対照実験（with / without skill）
指標改善
通過率100%（両グループ同一）
修正ポイント+36%
検証回数+65%
ツール呼び出し+50%
隠れた問題の発見率+50%

設定レビューシナリオでは、without_skill が Redis 設定ミスと CORS ワイルドカードのセキュリティリスクを見逃した例がある。with_skill の「主体的行動チェックリスト」が表面的な修正を超えたレビューを促進。

6 — インストール
各環境

Claude Code は claude plugin marketplace add tanweai/pua または手動で ~/.claude/plugins/pua に clone。Codex CLI / Cursor / Kiro / OpenClaw / Antigravity / OpenCode は、いずれも Agent Skills オープンスタンダード（SKILL.md）に従い、対応ディレクトリへ SKILL ファイルと必要に応じてコマンド（/pua）を配置する。Cursor は .cursor/rules/pua-ja.mdc、Kiro は .kiro/steering/pua-ja.md または .kiro/skills/pua-ja/SKILL.md。詳細は公式リポジトリの README を参照。

# Claude Code
claude plugin marketplace add tanweai/pua
claude plugin install pua@pua-skills

# Cursor（プロジェクトの .cursor/rules にルールを配置）
curl -o .cursor/rules/pua-ja.mdc \
  https://raw.githubusercontent.com/tanweai/pua/main/cursor/rules/pua-ja.mdc
