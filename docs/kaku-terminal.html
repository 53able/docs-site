Kaku

A fast, out-of-the-box terminal built for AI coding

Deeply customized fork of WezTerm for macOS. Zero config, theme-aware, curated shell stack, and a smaller binary with instant startup. Full WezTerm Lua config compatibility.

github.com/tw93/Kaku

Features

Zero Config

Defaults: JetBrains Mono, opencode theme, macOS font rendering, low-res font sizing. Ready on first launch.

Theme-Aware

Built-in dark/light themes with tuned selection colors, font weight, and practical color_overrides support.

Curated Shell Suite

Built-in zsh plugins (z, completions, syntax highlighting, autosuggestions). Optional: Starship, Delta, Lazygit, Yazi.

Fast & Lightweight

~40% smaller binary, instant startup, lazy loading, stripped-down GPU-accelerated core.

WezTerm-Compatible

Use WezTerm's Lua config directly; full API compatibility, no migration.

Quick Start

  1. Download Kaku DMG and drag to Applications, or brew install tw93/tap/kakuku
  2. Open Kaku (Apple-notarized; no security warnings)
  3. First launch auto-sets up your shell environment
  4. Config: ~/.config/kaku/kaku.lua (WezTerm Lua API). Run kaku config or Cmd + , for Settings TUI

Kaku AI

Kaku Assistant analyzes failed commands and suggests a safe fix; press Cmd + Shift + E to apply. Configure via kaku ai or Cmd + Shift + A: enable/disable, model, base URL, API key. One place to manage external AI tools (Claude Code, Codex, Gemini CLI, Copilot CLI, etc.). For custom headers (e.g. enterprise gateway), edit ~/.config/kaku/assistant.toml and set custom_headers.

CLI

kaku          # list commands
kaku ai       # AI settings
kaku config   # open Settings TUI
kaku doctor   # diagnostics (Ctrl + Shift + L in app)
kaku update   # check for updates
kaku cli split-pane -- bash -c "echo Hello"
