Learn Claude Code

Anthropic 公式の AI コーディングアシスタント。コードベースの読解・編集・コマンド実行・開発ツール連携。

1 — 概要
何か

Claude Code（Learn Claude Code）は、Anthropic が提供する AI コーディングアシスタント。コードベースの読解・ファイル編集・ターミナルコマンド実行・IDE や CI/CD との連携を通じて、機能追加・バグ修正・リファクタリング・自動化を支援する。公式ドキュメントでは「Understanding codebases」「Finding relevant code」「Fixing bugs」「Refactoring code」「Using subagents」などのワークフローが案内されている。

CLI推奨
VS Code拡張
Desktopアプリ
Subagents専門委譲

利用には Claude のサブスクリプション（Pro / Max / Teams / Enterprise）、Claude Console アカウント、または対応クラウドプロバイダ経由のアクセスが必要。

2 — 利用形態
どこで動くか
形態説明
Terminal CLI推奨。フル機能のコマンドライン。macOS/Linux/WSL ではネイティブインストール、Homebrew / WinGet も利用可。
VS Code 拡張インライン diff・@ メンション・プラン確認・会話履歴。
デスクトップアプリスタンドアロン。ビジュアル diff 確認と複数セッション。
Webブラウザ上で実行。
その他JetBrains IDE・Slack・CI/CD ツールなどにも対応。
3 — よくあるワークフロー
公式チュートリアルより
  • Understanding codebases — コードベースの概要やアーキテクチャパターンの把握
  • Finding relevant code — 特定機能に関連するファイルの特定
  • Fixing bugs — エラー共有と修正案の取得
  • Refactoring code — レガシーコードの安全な近代化
  • Using subagents — コードレビュー・テストなど専門タスクの委譲
4 — 導入
ターミナル（macOS/Linux/WSL）

推奨のネイティブインストール:

curl -fsSL https://claude.ai/install.sh | bash

その他は Homebrew や WinGet からもインストール可能。詳細は公式 Quickstart を参照。

注意

Claude Code の利用には、Claude の有料プラン（Pro / Max / Teams / Enterprise）、Claude Console アカウント、または対応クラウドプロバイダ経由のアクセスが必要です。
