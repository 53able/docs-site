AltSendme

alt-sendme — free, open-source P2P file transfer / Tauri + Iroh / v0.3.4

1 — 概要
何か

AltSendme は、クラウドにファイルを保存せず、デバイス間で直接ファイルを転送するオープンソースのデスクトップアプリ。 WeTransfer や Google Drive の代わりに、Iroh（P2P ライブラリ）を使って QUIC + TLS 1.3 で暗号化した P2P 転送を実現する。 アカウント登録不要、サイズ制限なし、転送中もファイルはサーバーを通過しない。

3 — 転送の仕組み
送受信フロー

ユーザーから見た操作はシンプル 3 ステップ。内部では Blob・Ticket・Iroh endpoint の 3 要素が連携する。

1. ファイル/フォルダをドロップ → 2. Ticket（接続情報）を生成 → 3. Ticket を相手に共有 → 4. 受信者が貼り付け → 5. 転送開始
Blobs — コンテンツアドレス型ストレージ

iroh-blobs は任意のバイト列（Blob）を BLAKE3 ハッシュで識別する。ハッシュが一致すれば内容が同じことを保証でき、中断・再開時のチェックポイントにも使われる。

概念説明
Blobメタデータを持たない生のバイト列。ファイル 1 個 = Blob 1 個。
Link（Hash）Blob を一意に識別する 32 バイトの BLAKE3 ハッシュ。
HashSeq複数 Blob の Link を並べた Blob。ディレクトリ転送で使用。
Provider / Requester送信側が Provider、受信側が Requester。1 endpoint が両方を兼ねることも可能。
Tickets — 接続情報トークン

Ticket はエンドポイントに接続するために必要なすべての情報（アドレス・NodeId・Blob のハッシュ）を 1 つのトークンに詰め込んだもの。チャット・メール・SMS で貼り付けるだけで転送が始まる。

プライバシーの優位性: 完全な P2P（BitTorrent 方式）では IP アドレスを不特定のピアにブロードキャストするが、Iroh の Ticket 方式は「明示的に接続したい相手とだけ Cozy Network を形成」するため IP が漏れない。
4 — ネットワーク層
QUIC + TLS 1.3

QUIC は UDP ベースの次世代トランスポートプロトコル（HTTP/3 の基盤）。TCP に比べレイテンシが低く、TLS 1.3 を直接統合している。AltSendme が QUIC を選んだ理由：

  • ゼロ RTT 接続確立 — 同じエンドポイントと再接続するとき 0-RTT が使える。
  • ストリーム多重化 — 複数ファイルを並列転送してもヘッドオブラインブロッキングが起きない。
  • 前向き秘匿性 — セッション鍵が過去の通信を復号するのに使えない。
  • 暗号化データグラム — 信頼性不要の小さなデータも暗号化して送れる。
NAT 越え・ホールパンチング

多くのデバイスはルーターの NAT 配下にあり、直接接続できない。Iroh はスタートアップ時にパブリックリレーサーバーに登録し、NAT・ファイアウォールを越えて接続を確立する。 接続できたあとは QUIC ホールパンチングで直接 P2P パスを確立し、リレーをバイパスする。

起動 → Relay 登録 → Ticket 交換 → QUIC ホールパンチ試行 → Direct P2P 確立
Relay サーバー（フォールバック）

直接 P2P 接続が不可能な場合のみ、オープンソースのリレーサーバーが暗号化パケットを中継する。 リレーはあくまで「暗号化された UDP パケットを転送するだけ」でコンテンツを復号できない。 自前のリレーをセルフホストすることも可能（ロードマップ項目）。

# クローン & 依存インストール
git clone https://github.com/tonyantony300/alt-sendme.git
cd alt-sendme
npm install
cargo install tauri-cli

# 開発サーバー（ホットリロード付き）
cargo tauri dev

# デスクトップバイナリのビルド
cargo tauri build --no-bundle

# Android エミュレータで起動
cargo tauri android dev

# Lint / 型チェック
npm run lint
ローカルテスト: Sendme CLI をインストールすると、同一デバイス内で送受信をテストできる（ファイルはデバイスの外に出ない）。
