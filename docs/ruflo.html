何をするか

Ruflo（旧称 Claude Flow）は、Claude Code に「神経系」を足す OSS スタック。エージェントのスウォーム化、タスクからの学習、セッションをまたいだ記憶、他ホストのエージェントとの安全な連携までをプラグインとフックで束ねる。

ポリシー・埋め込み・証明系などは Rust で書かれた WASM カーネルが担当する、という位置づけが README で強調されている。利用者は通常どおり Claude Code でコードを書き、バックグラウンドでルーティングと調整が走る想定。

Entry

名前の由来（README 抜粋）: 「Ru」は作者の「Ruv」、「flo」は flow。フロー状態と Rust 基盤を掛け合わせたブランド。

4 — クイックスタート
Claude Code プラグイン（推奨）

README のコマンドをそのまま掲載。環境に合わせてプラグインの組み合わせを選ぶ。

# Add the marketplace
/plugin marketplace add ruvnet/ruflo

# Install core + any plugins you need
/plugin install ruflo-core@ruflo
/plugin install ruflo-swarm@ruflo
/plugin install ruflo-autopilot@ruflo
/plugin install ruflo-federation@ruflo
