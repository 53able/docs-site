Palmier Pro

palmier-io/palmier-pro — Swift ネイティブの macOS 動画エディタ。タイムライン編集と生成 AI を同一プロジェクト内で扱い、Claude / Cursor / Codex から MCP で操作できる。

プロジェクト概要

Palmier Pro は、人間とエージェントが同じタイムライン上で動画を生成・編集するためのオープンソース動画エディタ。Premiere Pro を参照モデルにしつつ、生成 AI と MCP 連携をワークフローに組み込む。

1 — 概要
解決する課題

YC S24 の Palmier が社内で AI 動画制作を行う中で、Web 上の生成プラットフォームとデスクトップのタイムラインエディタの往復がボトルネックになっていた。生成 → ダウンロード → インポート → 差し替え編集のループと、バージョン管理の煩雑さが主な痛み。

Palmier Pro はタイムラインを単一のソース・オブ・トゥルースに据え、スクリプト作成・生成・編集のコンテキストを一か所に集約する。エージェントはアプリ内チャットまたは外部 MCP クライアントから同じツール群にアクセスできる。

対象プラットフォーム: macOS 26 (Tahoe) on Apple Silicon のみ。ログイン不要でエディタ本体と MCP サーバーは無料利用可能。生成 AI 機能はログインとサブスクリプションが必要。
2 — 主要機能
3 つの柱
01
Swift ネイティブ動画エディタ
ゼロから Swift で構築。タイムライン・プレビュー・メディアパネル・エクスポートなど、従来型 NLE の骨格を持つ。エフェクト・トランジション・カラーグレード等は未実装。
02
組み込み生成 AI
Seedance、Kling、Nano Banana Pro などの SOTA モデルでタイムライン内から動画・画像を生成。処理パイプラインはクローズドソース。
03
エージェント連携 (MCP)
アプリ起動中に http://127.0.0.1:19789/mcp で HTTP MCP を公開。タイムライン読み取り・クリップ編集・メディア生成・トラック操作を外部エージェントから実行可能。
04
mcpb バンドル
Claude Desktop 向け Desktop Extension を同梱。mcp-remote プロキシ経由でローカル HTTP サーバーへ接続する Node エントリを mcpb/ に配置。

4 — MCP 接続
前提

Palmier Pro を起動し、プロジェクトを開いた状態で http://127.0.0.1:19789/mcp が応答する。Claude Desktop は mcpb/ バンドル（mcp-remote プロキシ）経由で同じエンドポイントへ接続する。

Claude Code

bash
claude mcp add --transport http palmier-pro http://127.0.0.1:19789/mcp

Codex

bash
codex mcp add palmier-pro --url http://127.0.0.1:19789/mcp

Cursor

アプリ内 Help → MCP Instructions → Install in Cursor が最短。手動設定は ~/.cursor/mcp.json に以下を追加:

json
{
  "mcpServers": {
    "palmier-pro": {
      "type": "http",
      "url": "http://127.0.0.1:19789/mcp"
    }
  }
}

Claude Desktop（mcpb）

Help → MCP Instructions → Install in Claude Desktop で Desktop Extension をワンクリック導入。バンドル内の Node エントリは mcp-remote を起動し、ローカル HTTP MCP へプロキシする:

mcpb/server/index.js（要約）
process.argv = [
  process.argv[0], 'mcp-remote',
  'http://127.0.0.1:19789/mcp',
  '--allow-http', '--transport', 'http-only'
];
require('mcp-remote/dist/proxy.js');
5 — エージェント連携
MCP サーバー vs アプリ内チャット

同一のプロンプトとツール群を共有するが、UX と課金モデルが異なる。

観点外部 MCP（Claude / Cursor / Codex）アプリ内チャット
料金無料（MCP サーバー自体）BYOK またはサブスクリプション
メディア参照ツール経由@ でメディアを直接参照（生成イテレーション向き）
コンテキストクライアント側のメモリ・Web 検索を活用タイムライン直上でコンテキストウィンドウを制御
連携例Slack MCP + Epidemic Sound MCP と併用編集作業中のコンテキスト切替が少ない
対応モデルMCP 経由で各クライアントのモデル現状 Anthropic のみ（アプリ内）

MCP 連携のユースケース例

  • Claude で脚本を書き、同じ会話から Palmier Pro 内で動画を生成する
  • Epidemic Sound MCP で効果音を取得し、Palmier Pro MCP でタイムラインに配置する
  • Slack の #marketing からアイデアを取得し、プロトタイプ動画を素早く組む

7 — 開発
ビルド手順

前提: macOS 26+、Xcode 16+、Swift 6.2 toolchain。

bash — 基本ビルド
git clone https://github.com/palmier-io/palmier-pro
cd palmier-pro
swift build
swift run

デバッグ用 .app バンドルと OSLog ストリーム:

bash — scripts/dev.sh
./scripts/dev.sh
コントリビューション: 大規模な無依頼 PR はレビュー帯域の都合で受けにくい。Issue ベースのバグ報告・機能要望が推奨される（CONTRIBUTING.md）。
