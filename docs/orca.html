Orca

stablyai/orca — 並列エージェント開発向け AI オーケストレータ（Electron IDE）

stablyai/orca は、Codex / Claude Code / OpenCode / Pi などターミナルで動く CLI エージェントを、それぞれ独立した git worktree で並べて走らせ、1 画面で追跡するデスクトップアプリ。モバイル companion・埋め込みブラウザ（Design Mode）・SSH worktree・スクリプト可能な orca CLI を同梱する。MIT。サイトは onorca.dev。

機能
README の主要機能
Parallel Worktrees
1 プロンプトを複数エージェントへ展開。各々が隔離された git worktree。結果比較して勝者をマージ。
Mobile Companion
スマホから監視・追指示。完了通知。iOS App Store / TestFlight / Android APK。
Terminal Splits
WebGL レンダリング、無制限分割、再起動後も残るスクロールバック（Ghostty 級を標榜）。
Design Mode
埋め込み Chromium で要素クリック → HTML / CSS / 切り抜きスクショをエージェントへ。
GitHub & Linear
PR・Issue・ボードをアプリ内で閲覧。タスクから worktree を開いてレビュー。
SSH Worktrees
リモート箱でエージェント実行。ファイル編集・git・端末、再接続とポートフォワード付き。
Annotate AI Diffs
diff 行コメントをエージェントへ戻す。レビュー〜コミットをアプリ内で完結。
Orca CLI
orca worktree create / snapshot / click / fill などでワークフローをスクリプト化。

その他: Quick open、アカウント切替と usage 追跡、Markdown/PDF プレビュー、Computer Use、通知 / unread。

構成
プロセス境界

Electron + electron-vite。主要ディレクトリは src/main（本体）・src/preload・src/renderer（UI / shadcn）・src/cli・src/relay・src/shared。モバイルは mobile/。エージェント向けスキルは skills/。

flowchart TD
  UI[renderer UI] --> PRE[preload]
  PRE --> MAIN[main process]
  MAIN --> WT[worktrees + terminals]
  MAIN --> BR[embedded browser]
  MAIN --> REL[relay / SSH]
  CLI[orca CLI] --> MAIN
  MOB[mobile companion] --> MAIN
  WT --> AG[CLI agents in PTYs]
  SK[skills: orca-cli / orchestration] --> CLI

CLI とオーケストレーション
orca CLI

Orca が起動中のランタイムが正。エージェントから worktree / terminal / 埋め込みブラウザを操作するときは raw git worktree や外付け Playwright より CLI を優先（スキル skills/orca-cli）。

# Linux では端末外の bare `orca` が GNOME スクリーンリーダーのことがある
command -v orca-ide || command -v orca
orca status --json
orca worktree ps --json
orca terminal list --json

# 独立 worktree へのフルハンドオフ例
orca worktree create --name <task> --no-parent --agent codex --prompt "<brief>" --json

worktree id は <repoId>::<worktreePath> の二部アドレス。JSON の id をそのまま次コマンドへ渡す。

orchestration vs フルハンドオフ
用途使うもの要点
監視付きマルチエージェント（DAG・ask/reply・decision gate） orca orchestration … task-create / dispatch --inject / check --wait。provenance 必須。
所有権を渡して元エージェントは終了 orca worktree create 等 「hand off」系。orchestration の task 行は作らない。
埋め込みブラウザ操作 orca CLI（click / fill 等） 外のデスクトップ UI は Computer Use スキル側。
