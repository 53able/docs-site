CompressO

動画・画像をローカルで圧縮する Tauri デスクトップアプリ（解説元: repos/compressO）

1 — 概要
位置づけ

CompressO（読みは Espresso に近い）は、動画・画像をオフラインで小さくするための無料オープンソースアプリ。フロントは React + Vite、シェルは Tauri 2、実際のエンコードは FFmpeg・pngquant・jpegoptim・gifski などプラットフォーム別のスタンドアロンバイナリが担当する（codeforreal1/compressO）。

Tauri 2デスクトップ
React 18UI
Offline通常時ネット不要
AGPL 3.0ライセンス

アプリ本体からの通信は原則なく、組み込みアップデート用途を除く。トリム・分割・バッチ・字幕・メタデータなど README 上の機能はフロント＋ Tauri コマンド＋ 外部ツールの組み合わせで実現している。

2 — アーキテクチャ
レイヤ構造

UI は TanStack Router でルート分割し、状態は Valtio。ネイティブ処理は Rust 側（src-tauri）に集約し、TypeScript から invoke で呼び出す。圧縮パイプラインは FFmpeg 等の sidecar を Rust が起動・制御するイメージ。

4 — 開発・ビルド
よく使うコマンド

前提: Rust と Node.js、パッケージマネージャは pnpm（package.json）。

# Tauri 開発（アプリ全体）
pnpm tauri:dev

# フロントのみ（ポート 3001）
pnpm vite:dev

# 本番ビルド
pnpm tauri:build

# TS 型チェック
pnpm tsc:check

# Rust（src-tauri）
cd src-tauri && cargo clippy

6 — 注意・ライセンス
macOS: 開発者未検証・未 notarize のため Gatekeeper 警告が出る場合がある。README では xattr -cr /Applications/CompressO.app や Homebrew 利用が案内されている。
