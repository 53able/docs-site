VoltAgent

AI Agent Engineering Platform — TypeScript フレームワークと VoltOps Console

1 — 概要
位置づけ
@voltagent/core MIT pnpm / Lerna Node 20+

VoltAgent は、エージェントをコードで組み立てて本番運用までつなぐためのプラットフォーム。大きく オープンソースの TypeScript フレームワーク と VoltOps Console（クラウド / セルフホスト）の2つで構成される。

フレームワーク側ではメモリ・RAG・ガードレール・ツール・MCP・Voice・ワークフローなどを型付きで扱う。VoltOps 側ではトレース、ログ、デプロイ、Eval、プロンプト管理など運用面をカバーする。

MCP ドキュメントサーバ

モノレポでは packages/docs-mcp（npm 名 @voltagent/docs-mcp）。Claude / Cursor / Windsurf 等からドキュメント・例・changelog を参照させられる。README では @voltagent/mcp-docs-server と書かれることもある。セットアップは 公式ドキュメント。

4 — クイックスタート
create-voltagent-app

CLI でプロジェクトを生成し、src/index.ts にエージェントと src/workflows/index.ts のワークフロー例が入る。

npm create voltagent-app@latest
cd <project>
npm run dev

起動後、ターミナルに表示される VoltOps Console の URL からエージェントを確認・チャットできる（README の手順）。デフォルトの HTTP ポートは 3141。
