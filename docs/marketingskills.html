marketingskills

Corey Haines 氏の marketingskills — マーケ専用 Agent Skills の俯瞰

1 — 概要
何のリポジトリか

Marketing Skills for AI Agents は、Agent Skills 形式の Markdown スキルを、CRO・コピー・SEO・計測・GTM など マーケ課題に揃えて束ねたコレクション。各スキルは SKILL.md 主体で、references/・evals/ を併用。Claude Code、Codex、Cursor、Windsurf 等、スキル対応エージェント向けに設計されている。

38+ 独立スキル
PMC product-marketing-context 軸
MIT ライセンス
tools CLI レジストリ同梱

中核となる product-marketing-context がプロダクト・顧客・訴求の前提を握り、他スキルは 同じ文脈を先に参照してから実行する前提で相互参照する。README では SEO・CRO・リテンションなど 7 カラムの要約図と、copywriting ↔ page-cro ↔ ab-test-setup などのクロスリードも明示される。

4 — 導入
代表的な取り込み方

CLI（推奨）

公式 README は npx skills を最優先で案内。coreyhaines31/marketingskills から一括 or スキル単位で .agents/skills/ 等へ入る。

# すべて
npx skills add coreyhaines31/marketingskills

# 特定のみ
npx skills add coreyhaines31/marketingskills --skill page-cro copywriting

その他

  • Claude Code: /plugin marketplace add coreyhaines31/marketingskills から marketing-skills プラグイン
  • 生クローン: cp -r marketingskills/skills/* .agents/skills/ 等
  • git submodule: .agents/marketingskills として常時取り込み
  • SkillKit 経由: npx skillkit install coreyhaines31/marketingskills（マルチエージェント向け）
v1 からの移行: プロダクト文脈ファイルの置き場所が .claude/product-marketing-context.md から .agents/product-marketing-context.md 優先に変わった。未移行でも .claude/ フォールバックあり。
5 — 使い方
会話例

依頼文の例（README 抜粋）:

"Help me optimize this landing page for conversions" → page-cro
"Write homepage copy for my SaaS" → copywriting
"Set up GA4 tracking for signups" → analytics-tracking
"Create a 5-email welcome sequence" → email-sequence

各スキルに /page-cro 形式のショートカット前提が書かれていれば、エージェントのコマンド体系に合わせて呼び出しやすい。

6 — tools レジストリ
スキル外の補助データ

tools/REGISTRY.md は、GA4・HubSpot・Stripe など 外部マーケ／計測ツールについて、API / MCP / 参照 CLI / ガイド有無を一覧化した「エージェント向け道具箱インデックス」。tools/clis/*.js と tools/integrations/*.md を横断的に辿るための出発点。スキル本文とは独立しているが、連携作業（計測・配信・CRM）で参照しやすい形になっている。
