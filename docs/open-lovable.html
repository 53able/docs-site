Open Lovable

firecrawl/open-lovable — MIT。チャットで React アプリを生成するデモアプリ

1 — 概要
位置づけ

Open Lovable は Firecrawl チームが公開しているサンプル実装。AI と対話しながら React アプリのコードを生成・適用し、リモートサンドボックス上でプレビューする構成だ。クラウドの完成品としては Lovable.dev が別途ある、と README が案内している。

2 — 技術スタック（README・package より）
全体像

フロント／API は Next.js 15（next dev --turbopack）。生成まわりには Vercel ai パッケージと各種 @ai-sdk/*、スクレイプに @mendable/firecrawl-js。サンドボックスは @vercel/sandbox か @e2b/code-interpreter のどちらかを選択する。

コード側のデフォルト: lib/sandbox/factory.ts では SANDBOX_PROVIDER 未設定時のフォールバックが e2b。README では vercel を例示しているため、運用時は環境変数で明示するのが安全だ。

5 — 起動手順
README どおり 
git clone https://github.com/firecrawl/open-lovable.git
cd open-lovable
pnpm install

# .env.local を作成（上表を参照）
pnpm dev

ブラウザで http://localhost:3000。npm / yarn でも可、と README に記載がある。
