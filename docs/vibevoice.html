1 — 概要
VibeVoice とは

Microsoft が開発したオープンソースの フロンティア音声 AI モデルファミリー。音声認識（ASR）とテキスト音声変換（TTS）の両方向をカバーし、長時間音声の一括処理を核心的な設計思想に置く。

コアイノベーションは超低フレームレート 7.5 Hz で動作する連続音声トークナイザ（Acoustic + Semantic）。音声品質を保ちながら長シーケンスの計算効率を大幅に向上させ、60 分音声の 1 パス処理を実現する。

4 — ASR 詳細
VibeVoice-ASR の主要機能
60-Minute Single-Pass
長時間一括処理
短チャンク分割方式と異なり、最大 60 分（64K トークン）を 1 パスで処理。セッション全体の話者追跡と意味的一貫性を維持。
Who / When / What
リッチ書き起こし
ASR・話者分離・タイムスタンプを同時実行。「誰が」「いつ」「何を話したか」を構造化データとして出力。
Customized Hotwords
カスタムホットワード
特定の固有名詞・専門用語・背景情報をプロンプトとして与え、認識精度を向上。ドメイン特化コンテンツに有効。
50+ Languages
多言語 + コードスイッチ
50 言語超をネイティブサポート。言語設定不要で自動検出。文中・発話間のコードスイッチングも自然に処理。

LoRA ファインチューニング / vLLM 高速推論

LoRA（Low-Rank Adaptation）によるファインチューニングをサポート。ドメイン特化データで追加学習が可能。詳細は finetuning-asr/ を参照。

vLLM 対応: vllm_plugin で vLLM を通じた高速推論もサポート。大規模バッチ処理や本番環境でのスループット向上に有効。また 2026-03 以降、Hugging Face transformers ライブラリ（v5.3.0+）からも直接利用可能。
5 — クイックスタート
インストール 
# Docker 環境推奨 (NVIDIA PyTorch Container 24.07〜25.12)
sudo docker run --privileged --net=host --ipc=host \
  --ulimit memlock=-1:-1 --ulimit stack=-1:-1 \
  --gpus all --rm -it nvcr.io/nvidia/pytorch:25.12-py3

# リポジトリをクローン
git clone https://github.com/microsoft/VibeVoice.git
cd VibeVoice

# ASR / 基本インストール
pip install -e .

# Realtime TTS 用（transformers==4.51.3 が必要）
pip install -e .[streamingtts]
VibeVoice-ASR の使い方 
# Gradio デモを起動
apt update && apt install ffmpeg -y
python demo/vibevoice_asr_gradio_demo.py \
  --model_path microsoft/VibeVoice-ASR --share

# ファイルから直接推論
python demo/vibevoice_asr_inference_from_file.py \
  --model_path microsoft/VibeVoice-ASR \
  --audio_files /path/to/audio.wav
VibeVoice-Realtime の使い方 
# WebSocket デモ（NVIDIA T4 / Mac M4 Pro で実時間動作確認済み）
python demo/vibevoice_realtime_demo.py \
  --model_path microsoft/VibeVoice-Realtime-0.5B

# ファイルから直接推論（話者指定あり）
python demo/realtime_model_inference_from_file.py \
  --model_path microsoft/VibeVoice-Realtime-0.5B \
  --txt_path demo/text_examples/1p_vibevoice.txt \
  --speaker_name Carter

# 追加実験的話者のダウンロード（9 言語 + 11 英語スタイル）
bash demo/download_experimental_voices.sh
